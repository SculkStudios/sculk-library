[Java Guide](/getting-started/java/).

## Bootstrap

The simplest way to start is to extend `SculkPlugin`. It creates the platform for you, exposes it as
`sculk` (`getSculk()` in Java), and closes it on disable — no `onEnable`/`onDisable` boilerplate.

The constructor block enables **optional systems** that need platform-level setup:
- `gui()` — registers inventory click/close routing so GUI click handlers fire
- `config()` — enables the typed config system (load, reload, callbacks)
- `data()` — opens the database connection pool (SQLite by default)
- `text()` — loads localization bundles from `lang/`

**Commands, events, the scheduler, and tasks are always available** — no builder call needed.

<Tabs syncKey="lang">
<TabItem label="Kotlin">

```kotlin
class MyPlugin : SculkPlugin({ gui(); config() }) {
    override fun setup() {
        sculk.commands.registerAll(
            greetCommand(),
            homeCommand(),
        )
    }

    // Optional: flush/save before the platform closes.
    override fun shutdown() {}
}
```

</TabItem>
<TabItem label="Java">

```java
public final class MyPlugin extends SculkPlugin {
    public MyPlugin() {
        super(b -> {
            b.gui();
            b.config();
        });
    }

    @Override
    protected void setup() {
        getSculk().getCommands().registerAll(greetCommand(), homeCommand());
    }

    // Optional: flush/save before the platform closes.
    @Override
    protected void shutdown() {}
}
```

</TabItem>
</Tabs>

That's the whole bootstrap. Put registration in `setup()` and any teardown in `shutdown()`.

### Full control (manual)

If you need to own the enable/disable sequence yourself, extend `JavaPlugin` and create the platform
directly:

<Tabs syncKey="lang">
<TabItem label="Kotlin">

```kotlin
class MyPlugin : JavaPlugin() {
    lateinit var sculk: SculkPlatform

    override fun onEnable() {
        sculk = SculkPlatform.create(this) { gui(); config() }
        sculk.commands.registerAll(greetCommand(), homeCommand())
    }

    override fun onDisable(): Unit = sculk.close()
}
```

</TabItem>
<TabItem label="Java">

```java
public final class MyPlugin extends JavaPlugin {
    private SculkPlatform sculk;

    @Override
    public void onEnable() {
        sculk = SculkPlatform.create(this, b -> {
            b.gui();
            b.config();
        });
        sculk.getCommands().registerAll(greetCommand(), homeCommand());
    }

    @Override
    public void onDisable() {
        sculk.close();
    }
}
```

</TabItem>
</Tabs>

## A command in its own file

<Tabs syncKey="lang">
<TabItem label="Kotlin">

```kotlin
// commands/GreetCommand.kt
fun greetCommand() = command("greet") {
    description = "Greet the player."
    permission = "myplugin.greet"

    player {
        reply("<green>Hey, <white>${player!!.name}<green>!")
    }
}
```

</TabItem>
<TabItem label="Java">

```java
// commands/GreetCommand.java
static CommandBuilder greetCommand() {
    return SculkCommands.command("greet", cmd -> {
        cmd.setDescription("Greet the player.");
        cmd.setPermission("myplugin.greet");

        cmd.player(ctx -> {
            ctx.reply("<green>Hey, <white>" + ctx.getPlayer().getName() + "<green>!");
        });
    });
}
```

</TabItem>

// gui/MainMenu.kt
val mainMenu = gui("<dark_aqua><bold>Main Menu") {

</TabItem>
<TabItem label="Java">

```java
// gui/MainMenu.java
static Gui mainMenu(Gui settingsMenu) {
    return SculkGui.gui("<dark_aqua><bold>Main Menu", menu -> {
        menu.setSize(27);

        menu.item(13, slot -> {
            slot.setMaterial(Material.NETHER_STAR);
            slot.setName("<gold><bold>Settings");
            slot.lore("<gray>Click to open settings.");
            slot.onClick(ctx -> {
                ctx.open(settingsMenu);
            });
        });
    });
}
```

## Loading a config

Config models are plain data holders. Define the model once and load it from either language with the
same typed API.

<Tabs syncKey="lang">
<TabItem label="Kotlin">

```kotlin
@ConfigFile("settings.yml")
data class Settings(
    val prefix: String = "<gray>[<aqua>Plugin<gray>]",
    val maxHomes: Int = 5,
)

val settings = sculk.config.load<Settings>()
```

</TabItem>
<TabItem label="Java">

```java
// The model is defined once (Kotlin data class above). Load it from Java with a Class token:
Settings settings = getSculk().getConfig().load(Settings.class);
```

</TabItem>
</Tabs>
