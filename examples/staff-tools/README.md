# Staff Tools Recipe

Compile-checked moderation workflow using Sculk commands, events, GUI, and effects.

## Shows

- Permission-gated staff mode.
- Freeze and unfreeze command.
- Movement cancellation for frozen players.
- Staff chat recipient filtering.
- Read-only inspect menu.
- Audio and particle feedback.

## Commands

| Command | Permission | Purpose |
|---|---|---|
| `/staff` | `staff.mode` | Toggle staff mode. |
| `/freeze <player> [reason]` | `staff.freeze` | Freeze or unfreeze a player. |
| `/inspect <player>` | `staff.inspect` | Open inspection menu. |
| `/staffchat <message>` | `staff.chat` | Message online staff. |
