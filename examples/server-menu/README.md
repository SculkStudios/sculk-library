# Server Menu Recipe

Compile-checked example for a configurable main menu with real navigation.

## Shows

- GUI subsystem setup.
- Config-backed `ItemDescriptor` buttons.
- Warps menu with real teleport actions.
- Profile and online-player menus.
- Paginated GUI entries.
- Confirm menu pattern.

## Command

| Command | Purpose |
|---|---|
| `/menu` | Opens the server menu. |
